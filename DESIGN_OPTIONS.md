# EMSaide 应用设计方案

## 当前状态

你已经成功添加了邮箱账户，但不知道如何给其他人发邮件。

## 使用方法（当前版本）

### 1. 添加你自己的邮箱
- 这是第一步，你已经完成了
- 可以添加多个：QQ 邮箱、163 邮箱、Gmail 等

### 2. 进入聊天界面
- 点击你添加的邮箱
- 进入该邮箱的聊天窗口

### 3. 给别人发邮件
```
┌─────────────────────────────────────┐
│ 收件人：[friend@gmail.com      ]    │  ← 输入对方邮箱
│ 主题：  [周末聚会              ]    │  ← 输入主题
│ 消息：  [周末有空吗？          ]    │  ← 输入内容
│                        [发送]       │
└─────────────────────────────────────┘
```

### 4. 查看往来邮件
- 你发送的邮件显示在右边（绿色背景）
- 对方回复的邮件显示在左边（白色背景）
- 长按"发送"按钮可以同步新邮件

## 设计说明

**当前设计：以你的邮箱为中心**

每个聊天对象 = 你的一个邮箱账户

优点：
- 管理多个邮箱很方便
- 每个邮箱的聊天记录独立
- 清楚知道用哪个邮箱发的邮件

缺点：
- 不能像微信一样看到联系人列表
- 需要手动输入收件人邮箱

## 下一步改进方案

我可以实现以下两种方案之一：

### 方案 A：完全重构为微信模式
- 主界面显示联系人列表
- 每个联系人是一个聊天对象
- 发消息时选择用哪个邮箱发送

**工作量**: 大（需要重构整个应用）
**时间**: 较长

### 方案 B：保持当前设计，添加快捷功能
- 主界面仍然是邮箱列表
- 添加"常用联系人"功能
- 输入收件人时可以快速选择
- 自动记录发过的邮箱地址

**工作量**: 小（基于现有代码改进）
**时间**: 短

## 我的建议

**先使用当前版本**，体验一下基本流程：
1. 添加你的邮箱
2. 进入聊天界面
3. 输入朋友邮箱发一封测试邮件
4. 看看是否符合你的需求

如果觉得不够好用，告诉我具体哪里不方便，我会针对性改进。
