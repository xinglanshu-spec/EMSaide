# EMSaide 使用指南 - 如何与邮箱聊天

## 📧 应用的核心概念

EMSaide 将每个邮箱账户变成一个"聊天对象"，你可以：
- **接收邮件** = 收到对方的消息
- **发送邮件** = 给对方回复消息
- **邮件历史** = 聊天记录

## 🚀 完整使用流程

### 第一步：添加邮箱账户

1. 打开应用，点击右下角 **"+"** 按钮
2. 选择邮箱服务商（如 QQ 邮箱、Gmail 等）
3. 填写邮箱地址和**授权码**（不是登录密码！）
4. 服务器设置会自动填充
5. 点击 **"测试连接"** 验证配置
6. 点击 **"保存"** 添加账户

> ⚠️ **重要提示**：
> - QQ 邮箱、163 邮箱等需要使用**授权码**而非登录密码
> - 授权码在邮箱设置 → POP3/IMAP/SMTP 服务中获取
> - Gmail 可能需要特殊网络环境

### 第二步：查看收到的邮件（聊天消息）

1. 在主界面点击已添加的邮箱账户
2. 进入聊天界面，可以看到该邮箱收到的所有邮件
3. 邮件按时间顺序显示，类似聊天记录
4. **长按发送按钮**可以同步新邮件

### 第三步：发送邮件（回复消息）

这是关键！你需要填写以下信息：

#### 输入框说明

```
┌─────────────────────────────────────┐
│ 收件人：recipient@example.com       │  ← 要发送到的邮箱地址
│ 主题：Re: 昨天的会议                │  ← 邮件主题（可选）
│ 消息：你好，我想咨询一下...         │  ← 邮件正文内容
│                        [发送]       │  ← 点击发送邮件
└─────────────────────────────────────┘
```

#### 具体步骤

1. **收件人**：输入对方的邮箱地址
   - 例如：`friend@gmail.com`
   - 可以是任何邮箱地址

2. **主题**：输入邮件主题（可选）
   - 例如："Re: 合作事宜"、"问候"
   - 不填会默认使用"聊天消息"

3. **消息内容**：输入邮件正文
   - 可以写多行文字
   - 这是邮件的主要内容

4. **点击发送**：
   - 邮件会通过 SMTP 发送到对方邮箱
   - 发送记录会保存在聊天历史中
   - 绿色背景表示已发送的消息

## 💡 常见使用场景

### 场景 1：回复工作邮件

1. 点击公司邮箱账户
2. 看到老板发来的邮件
3. 在收件人栏填入老板的邮箱（或点击回复自动填充）
4. 主题自动加上 "Re: "
5. 写下你的回复
6. 点击发送

### 场景 2：与朋友用邮件聊天

1. 添加你的个人邮箱
2. 进入聊天界面
3. 收件人输入朋友的邮箱
4. 像发微信一样写信
5. 朋友收到后用邮件回复
6. 你再次同步就能看到回复

### 场景 3：管理多个邮箱

- 为每个邮箱添加独立的聊天对象
- 工作邮箱、个人邮箱、学校邮箱分开管理
- 每个邮箱的聊天记录独立保存
- 一键切换不同邮箱

## ⚙️ 功能详解

### 同步邮件（接收新消息）

**长按发送按钮** → 确认同步 → 从服务器下载新邮件

- 新邮件会显示在聊天窗口
- 下载的邮件会从服务器删除（只保存在本地）
- 可以多次同步，不会丢失数据

### 发送邮件（发送消息）

**填写收件人 + 主题 + 内容** → 点击发送 → 通过 SMTP 发送

- 邮件会发送到对方邮箱
- 发送记录保存在本地聊天历史
- 对方可以用任何邮箱客户端回复

### 查看历史

- 所有收到的邮件自动保存
- 所有发送的邮件自动保存
- 卸载应用会丢失所有数据（请谨慎！）

## 🔒 重要注意事项

### 关于授权码

大多数邮箱服务商要求使用授权码而非真实密码：

| 邮箱服务商 | 获取方式 |
|-----------|---------|
| QQ 邮箱 | 设置 → 账户 → POP3/IMAP/SMTP → 生成授权码 |
| 163 邮箱 | 设置 → POP3/SMTP/IMAP → 开启服务并获取授权码 |
| Gmail | Google 账户 → 安全性 → 应用专用密码 |
| Outlook | 账户安全 → 应用密码 |

### 关于邮件删除

- **读取即删除**：从服务器下载邮件后会自动删除
- **本地保存**：所有邮件保存在手机 SQLite 数据库
- **备份建议**：重要邮件请提前备份到其他位置
- **卸载警告**：卸载应用会丢失所有聊天数据

### 关于网络

- 需要稳定的互联网连接
- Gmail 等服务可能需要特殊网络环境
- 防火墙可能阻止某些邮件服务器

## ❓ 常见问题

### Q1: 我能给任何人发邮件吗？
**A**: 是的！只要知道对方的邮箱地址，就可以发送给任何人。

### Q2: 对方必须也用这个应用吗？
**A**: 不需要！对方可以用任何邮箱客户端（Outlook、Gmail App 等）回复你。

### Q3: 为什么我收不到邮件？
**A**: 检查以下几点：
- 网络连接是否正常
- 邮箱配置是否正确
- 是否开启了 POP3/IMAP服务
- 授权码是否正确

### Q4: 为什么发送失败？
**A**: 可能的原因：
- SMTP 服务器配置错误
- 端口或 SSL 设置不对
- 邮箱服务商限制了发送
- 网络连接问题

### Q5: 可以同时和多人聊天吗？
**A**: 当前版本每个邮箱账户是一个聊天对象。你可以：
- 给不同的人发送邮件
- 但所有对话都显示在同一个聊天窗口
- 未来版本可能会改进这一点

## 🎯 快速开始示例

假设你想用 QQ 邮箱给朋友发邮件：

1. **添加账户**
   ```
   邮箱：123456@qq.com
   授权码：abcdefghijklmnop（在 QQ 邮箱设置获取）
   ```

2. **进入聊天**
   - 点击 "123456@qq.com" 进入聊天界面

3. **发送邮件**
   ```
   收件人：friend@gmail.com
   主题：周末聚会
   内容：嗨！周末有空出来聚聚吗？
   ```

4. **等待回复**
   - 朋友收到邮件后用 Gmail 回复
   - 你长按发送按钮同步邮件
   - 看到朋友的回复显示在聊天窗口

就是这么简单！🎉
