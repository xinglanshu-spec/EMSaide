# EMSaide 更新说明 - 联系人模式

## 🎉 重大更新

应用已经重构为**以联系人为中心**的设计，现在可以像微信一样管理聊天对象了！

## ✨ 新功能

### 1. 联系人列表作为主界面
- 打开应用首先看到的是你的联系人列表
- 每个联系人是一个独立的聊天对象
- 类似微信的聊天列表体验

### 2. 添加联系人
- 点击右下角 "+" 按钮添加新联系人
- 填写联系人信息：
  - **名称**：显示在列表中的名字
  - **邮箱地址**：对方的邮箱
  - **备注**：可选，如"同事"、"朋友"

### 3. 与联系人聊天
- 点击联系人进入聊天界面
- **收件人自动填充**：自动填入该联系人的邮箱
- 只需填写主题和内容即可发送

### 4. 侧边栏菜单
- 点击左上角菜单图标打开侧边栏
- 可以访问：
  - 联系人（主界面）
  - 邮箱账户（管理你的邮箱）
  - 邮箱设置（配置 POP3/SMTP）

## 📱 使用流程

### 第一步：添加你的邮箱（如果还没添加）
1. 点击左上角菜单图标
2. 选择"邮箱设置"
3. 添加你的邮箱账户（QQ、163、Gmail 等）

### 第二步：添加联系人
1. 返回主界面
2. 点击右下角"+"按钮
3. 填写联系人信息
4. 点击"保存"

### 第三步：给联系人发邮件
1. 在主界面点击联系人
2. 进入聊天界面（收件人已自动填充）
3. 填写主题和内容
4. 点击"发送"

### 第四步：查看往来邮件
1. 你发送的邮件显示在右边（绿色背景）
2. 对方回复后，长按"发送"按钮同步邮件
3. 收到的邮件显示在左边（白色背景）

## 🔄 与旧版本的区别

### 旧版本（邮箱列表）
```
主界面
├─ your@qq.com
│  └─ 可以给任何人发邮件
└─ your@163.com
   └─ 可以给任何人发邮件
```

### 新版本（联系人列表）
```
主界面
├─ 张三 (zhangsan@gmail.com)
│  └─ 用你的某个邮箱给他发信
├─ 李四 (lisi@outlook.com)
│  └─ 用你的某个邮箱给她发信
└─ [添加联系人]
```

## 💡 提示

1. **多邮箱支持**：未来版本将支持选择用哪个邮箱发送邮件
2. **自动记录**：发送过的邮箱地址会自动保存为联系人
3. **一一对应**：每个联系人是一个独立的聊天对话
4. **数据迁移**：旧版本的聊天记录不会丢失，但需要手动关联到联系人
